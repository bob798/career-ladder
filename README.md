# career-ladder · 互联网公司岗位升级打怪路径

> 一套面向**个人学习与职业规划**的互联网岗位成长地图。
> 不做"网上普通学习路线图"，而是把**权威公司的职级框架、资深从业者的洞察**，提炼成一条条"升级打怪"的成长路径——帮你定位自己在哪一级、看清差距、知道下一步该练什么。

## 这是什么

每条"岗位之路"都不是技能清单的堆叠，而是一个**自洽的成长模型 + 权威来源校准**：

- **全景地图**：一条职级阶梯（升级路径）
- **暗线**：贯穿全程、决定"为什么能升级"的几条底层主线
- **逐级详解**：每一级统一模板（定位 / 核心工作 / 能力项 / 要打的"怪" / 晋级分界点）
- **定位与规划工具**：自测清单 + 差距诊断 + 下一步学习任务

## 怎么读

1. 先读开篇，记住那条**主命题**（它是全文的尺子）。
2. 看**全景地图**，找到自己当前的位置。
3. 用**暗线**理解"为什么能升级"的底层逻辑。
4. 翻到自己所在级别，重点看「要打的怪/失败形态」和「晋级分界点」。
5. 用文末的**定位与规划工具**做自测和下一步规划。

两篇刻意**同构**，可以对照阅读：它们讲的其实是同一件事——*在越来越大的约束/影响范围内，为越来越多的人，把事情做到位*。

## 方法论与引用

- 内容均用**自己的话重述**，不搬运原结构图、不全文翻译；英文短句为原文引用并标注出处。
- 来源分层使用：**公司职级框架**用于定义级别与影响范围；**资深从业者著作/文章**用于解释行为模式；**工程实践资料**用于补技术底盘。
- 各篇文末附完整来源清单与时效性提醒。
